# Port Scanner

A multithreaded TCP port scanner inspired by Nmap, built in Python.

## Features
- Multithreaded scanning (Lock & Semaphore)
- Service detection (SSH, HTTP, FTP, SMTP, MySQL...)
- Host availability check
- Two scan modes : specific ports or port range
- Persistent file logging
- GUI interface (tkinter)

## How to run
pip install tkinter
python main.py

## Files
- main.py  → entry point
- scan.py  → scanning logic
- gui.py   → graphical interface
